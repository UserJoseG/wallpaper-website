Vocabulary List

HTML: Hypertext Markup Language

HTML is the standard markup language for Web pages. With HTML you can create your own Website. HTML is easy to learn - You will enjoy it!

element: the technical term for the parts that make up a webpage such as text, links, and images. tag: the keywords or commands in an HTML document that create the elements on the webpage.

Webpages consist of a series of elements that tell the browser how to display the content. HTML elements are made from HTML tags that label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

file extension: The suffix at the end of a computer file name. It comes after the period and is usually two to four characters long.

File extensions are used by the operating system to identify what apps are associated with what file types—in other words, what app opens when you double-click the file. For example, a file named “awesome_picture.jpg” has the “jpg” file extension. When you open that file in Windows, for example, the operating system looks for whatever app is associated with JPG files, opens that app, and loads the file.

file path: describes the location of a file in a hard drive or web site's folder structure

File paths specify the location of individual files. They are used to give files access to one another and they are of two types : Absolute and Relative. Absolute file paths specify the location from the root folder of the hard drive or domain. Ex. C:\MyProject\doc.txt or domain.com/folder/index.html Relative file paths specify the location relative to the current folder. Ex. doc.txt or folder/page.html
